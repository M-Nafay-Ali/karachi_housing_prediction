Machine Learning Project (2nd Year CS Student)
This project is an exploratory data analysis and predictive modeling of the Karachi housing market. Using historical data from various neighborhoods (Clifton, North Nazimabad, Gulshan, etc.), I built a model to estimate house prices based on location, size, and room count.


Project Goal:-
As a student interested in Data Science, I wanted to understand:
Which areas in Karachi have the highest "Price per Square Meter"?
How strongly do factors like the number of bedrooms affect the total price?
Can we predict the market value of a house with reasonable accuracy using Linear Regression?

Key Features:-
Feature Engineering: Used Label Encoding to convert Karachi neighborhood names into numeric data for the model.
Correlation Analysis: Created a Seaborn Heatmap to visualize the relationship between house size and final cost.
Predictive Modeling: Implemented a Linear Regression algorithm to predict property prices.
Interactive Visuals: Used Plotly to create a bar chart showing the most expensive areas in the city.


Tech Stack:-
Language: Python
Platform: Google Colab (Mobile-friendly development)
Libraries: Pandas, NumPy, Seaborn, Matplotlib, Scikit-Learn, Plotly


Key Insights:-
The Location Factor: Area has the highest correlation with price, proving that in Karachi, where you buy is more important than what you buy.
Size vs. Price: A clear linear relationship was found between Size_sqm and Total_Price_Local.


Dataset Columns:-
Area: The neighborhood in Karachi.
Year: Year of the listing.
Price_per_sqm_Local: Value of the land per unit.
Size_sqm: Total area of the house.
Bedrooms: Number of rooms.
Total_Price_Local: (Target Variable) The final price of the house.
